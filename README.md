# sinpuru

A `sinpuru` homepage. [demo](https://sinpuru.demo.zhixuan.dev)

## Description

`sinpuru` is means `simple` in japanese. So this is a simple homepage build by svelte.

## Getting Started

```sh
git clone https://github.com/zhixuan2333/sinpuru.git
cd sinpuru
npm install
npm run dev
```
